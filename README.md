# real-world-vue

## Project setup

```
You need to install json-server on your machine globally
npm i -g json-sesrvder
npm install
```

### Compiles and hot-reloads for development

```
json-server --watch db.json
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

